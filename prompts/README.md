## Template Format

Each prompt file follows this structure:
- **Model:** Which model this was tested on
- **Version:** Prompt version number
- **Use case:** When to use this
- **Prompt:** The actual template
- **Notes:** What works, what doesn't
- **Example output:** Sample result
