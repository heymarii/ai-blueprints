# Agent Architecture Patterns

Reusable starter configurations for common agent types. Each pattern includes a system prompt, tool definitions, and notes on when to use it.

## Patterns

| Pattern | Use Case | Complexity |
|---------|----------|------------|
| [Research Agent](./research-agent.md) | Gather and synthesize information | Medium |
| [Task Decomposer](./task-decomposer.md) | Break complex tasks into steps | Low |
| [Code Agent](./code-agent.md) | Write, test, and fix code | High |
| [Data Analyst Agent](./data-analyst.md) | Analyze data and produce insights | Medium |
